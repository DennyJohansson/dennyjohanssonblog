## This is a Hugo app
### Install
#### Submodule
After cloning repo the submodule git repo will be empty so to fix that:

```
git submodule init
git submodule update
```
#### Quick start
[quick start guide](https://gohugo.io/getting-started/quick-start/)

#### Run
`hugo server -D`

#### New post
`hugo new content/posts/<date-name>`
